Relax and Recharge Express

The Relax and Recharge Express website and mobile app were developed using JavaScript, HTML, React, React Native, and Firebase. The platform offers travelers a seamless way to browse services, check real-time availability of food, drinks, and sleeping pods, and make reservations or payments effortlessly. With Firebase enabling secure data storage and real-time updates, the solution is designed to be responsive, user-friendly, and accessible across devices, enhancing the overall travel experience.

In the Relax and Recharge Express project, Firebase played a pivotal role in managing the backend functionalities of both the website and mobile app. Hereâ€™s how Firebase was utilized:

Authentication: Firebase Authentication was implemented to securely manage user sign-ups, logins, and profile authentication. Travelers and administrators can access their respective accounts using email/password or third-party authentication providers.

Database Management: Realtime Database/Cloud Firestore was used to store and manage data dynamically, such as:

  • Menu items: Real-time updates on food and drink availability.
  • Sleeping pod availability: Ensuring accurate data for bookings.
  • User data: Managing user profiles, preferences, and booking histories.
Firebase ensures seamless synchronization across devices, allowing users to see updates instantly.
